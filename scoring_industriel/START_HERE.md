# 🚀 START HERE — Démarrage Rapide

## En 3 commandes :

```bash
pip install -r requirements.txt
streamlit run app.py
```

Ouvre ton navigateur sur → **http://localhost:8501**

---

## Ce que tu verras :

1. **Formulaire gauche** — 5 étapes, 8 modules, ~100 champs
2. **Dashboard droit** — Résultats en temps réel après calcul

## Comment tester :

- Remplis quelques champs dans les étapes 1 à 5
- Clique sur **"✅ Valider et Calculer"** à l'étape 5
- Les 3 indices + radar + zones critiques apparaissent

## Pour un test rapide avec des données complètes :

Consulte `data/sample_input.json` — c'est l'exemple **AutoMaro SA** avec toutes les valeurs pré-remplies.
